# 📒 Contact Book V2

A simple terminal-based Contact Book built with Python. This project allows users to manage contacts by adding, viewing, and removing them while storing data permanently in a JSON file.

## ✨ Features

- 📋 View all saved contacts
- ➕ Add new contacts
- ❌ Remove existing contacts
- 💾 Save contacts in a JSON file for persistent storage
- 🖥️ Interactive terminal menu

## 🛠️ Built With

- Python 3
- JSON
- File Handling
- Dictionaries
- Functions

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Prasun01/Contact-Book-V2.git
```

2. Navigate to the project folder:

```bash
cd Contact-Book-V2
```

3. Run the application:

```bash
python Contact-book-v2.py
```

## 📖 Menu

```
1. View Contacts
2. Add Contact
3. Remove Contact
4. Exit
```
